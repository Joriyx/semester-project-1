# Community Science Museum

A website for a community science museum. Highlights upcoming events and practical information for prospective visitors.

Live [website](https://joriyx.github.io/semester-project-1/)

## Features

- Home page with links to exhibitions and events
- List of available exhibitions
- List of upcoming events
- Location and admission information

## Prerequisites

- Visual Studio Code
- Live Server (VS Code extension)

## Getting Started

### Running the project

Click the 'Go Live' button in VS Code
